# Phone-Report-IO
phone use report system for school staff.

# Reports record
  - # Reports
      - record of responses from the google form connected to the sheet.
# Reporting
  - # Most Reported
      - contains a list of the 10 students who have been reported the most, how many times they've been caught, and a form link pre-filled with thier name.
  - # Verified Reporters
      - list of emails that will have thier reports counted, other reports will be ignored but still logged in the Reports record.
  - # Report Form Link
      - contains the link to the report form that is referenced by the rest of the sheet.
  - # Studen Email
      - contains the student email domain.
  - # Settings
      - various miscellaneous settings for the rest of the sheet.
# Student report tracker
  - # Student Report Counter
      - contains an alphebetical by username table of all students usernames, school emails, parent emails, a pre-generated message for thier parents, and form link pre-filled with thier name.
# Students info
  - list of all student usernames and parent emails that is referanced by the rest of the sheet.
